# memebot

Meme bot is Super Blazingly Darn Fast as Light, written in Crystal :p (again)

## Installation

1. `git clone https://github.com/codenoid/Telegram-Bot-Boilerplate`
2. `cd Telegram-Bot-Boilerplate`
3. `crystal build src/memebot.cr`
4. `./memebot`
